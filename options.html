Scheduled Site Blocker – Config

Edit the JSON below and click Save. patterns accepts any match-pattern string. schedule is a list of [startHour, endHour] (24-h integers).


Example config

{
  "patterns": [
    "*://vnexpress.net/*",
    "*://*.facebook.com/*",
    "*://*.youtube.com/*",
    "*://goctruyentranhvui*"
  ],
  "schedule": {
    "mon": [
      [6, 12],
      [13, 23]
    ],
    "tue": [
      [6, 12],
      [13, 23]
    ],
    "wed": [
      [6, 12],
      [13, 23]
    ],
    "thu": [
      [6, 12],
      [13, 23]
    ],
    "fri": [
      [6, 12],
      [13, 23]
    ],
    "sat": [
      [6, 12],
      [14, 23]
    ],
    "sun": [
      [6, 12],
      [14, 23]
    ]
  }
}
